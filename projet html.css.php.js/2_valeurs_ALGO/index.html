Valeurs

Observez le code ci-dessous et tentez de prévoir les valeurs finales de chaque variable à la fin de chaque ligne (par exemple après la ligne 'var a = 2;' si je faisais un console.log de a j'aurai 2 comme résultat.
L'idée est de faire les calculs vous-mêmes, sur une feuille par exemple AVANT de regarder dans la console !

var a = 2;

a = a - 1;

a++;

var b = 8;

b += 2;

var c = a + b * b;

var d = a * b + b;

var e = a * (b + b);

var f = a * b / a;

var g = b / a * a;

Vérifiez vos prévisions en créant ce programme puis en le déboguant pour observer les valeurs des variables.
